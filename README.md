# nihonGO

A mobile-first Japanese language learning app powered by AI tutoring.

## About

nihonGO replaces traditional textbook learning with a personalized, interactive AI tutoring experience. The app focuses on practical Japanese acquisition through conversation, cultural context, and adaptive learning.

### Learning Philosophy

- **Cultural Context First** - Understand *why* natives choose specific expressions, not just grammar rules
- **Practical Examples** - Real-world scenarios over abstract textbook situations
- **Comparative Learning** - See differences between similar grammar points with native usage patterns
- **Personalized Experience** - Adapts to your background, goals, and preferred learning style

### Features

- **Learner Profile** - Customized learning based on your native language, prior experience, and goals
- **Study Mode** - In-depth grammar explanations with cultural context and multiple examples
- **Conversation Mode** - Quick, practical explanations with immediate dialogue practice
- **Vocabulary & Grammar** - Side-by-side comparisons with native usage examples

## Tech Stack

- **Framework**: React Native with Expo
- **Language**: TypeScript
- **Styling**: NativeWind (Tailwind CSS)
- **State Management**: Zustand
- **AI**: Claude API

## Getting Started

```bash
# Install dependencies
npm install

# Start the development server
npx expo start
```

## Environment Variables

Create a `.env` file in the root directory:

```
EXPO_PUBLIC_CLAUDE_API_KEY=your_api_key_here
```
